# covid-19_data-modeling
## overview
The goal of this project is to bulid a pipleine to convert Relational Data Model to Dimensional Data Model and store it in a data warehouse which can be used for visualization. This project analyzes a dataset related to COVID-19 using AWS tools. The dataset is stored in an S3 bucket and is accessed and analyzed using Athena, a serverless interactive query service. The data is transformed using Glue and is then loaded into a data warehouse Redshift for further analysis.
